JobQuest – Job Application Management Platform

Developed a job application management platform using Flask, PHP, RabbitMQ, and MySQL to streamline job search and tracking. Built the Flask front-end for user registration, job tracking, and networking features, while the PHP back-end processed job listings, user data, and core application logic. Integrated RabbitMQ for asynchronous communication between services, ensuring real-time updates and fault-tolerant operations. Designed a secure MySQL database to store user profiles, applications, and networking data, improving user experience and data reliability.

PenTest Report

Conducted penetration testing in a virtual lab using Kali Linux as the attacker system and Windows XP as the target. Tools like Metasploit, Nmap, and John the Ripper were used to identify vulnerabilities, including outdated software, weak password storage, and exposed services. Exploits demonstrated issues such as remote code execution via Internet Explorer (MS10-002 Aurora), cracking LM/NTLM password hashes, and detecting outdated services like Apache and FTP. Recommendations included patching software, enforcing strong password policies, and disabling unnecessary services to improve system security.
